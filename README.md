# LLM-Mafia

> [!NOTE]
> Draft document!

A classic Mafia game where each player is an LLM-backed Kubernetes pod. 

Players communicate via Kafka, reason with Ollama, and eliminated players get deleted from the cluster. The game is managed declaratively through Kubernetes resources.
