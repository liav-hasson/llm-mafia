# Learning Resources

Useful documentation and references i used during development.

## Golang

- [Go by Example](https://gobyexample.com/) — Learn Go through annotated examples
- [Go by Example - JSON](https://gobyexample.com/json) — Struct tags, Marshal/Unmarshal
- [Go Blog - JSON and Go](https://go.dev/blog/json) — Official deep dive on JSON handling
- [Effective Go - Embedding](https://go.dev/doc/effective_go#embedding) — Struct composition
- [encoding/json docs](https://pkg.go.dev/encoding/json) — Full API reference
- [time package docs](https://pkg.go.dev/time) — Time handling and Unix timestamps

## Kubernetes

- [Kubebuilder Book](https://book.kubebuilder.io/) — Official tutorial for building operators
- [Kubernetes CRDs](https://kubernetes.io/docs/concepts/extend-kubernetes/api-extension/custom-resources/) — Custom resource definitions

## Kafka

- [Kafka Introduction](https://kafka.apache.org/intro) — Core concepts: topics, partitions, consumer groups
- [Strimzi Docs](https://strimzi.io/documentation/) — Kafka operator for Kubernetes
- [kafka-go Library](https://github.com/segmentio/kafka-go) — Go client for Kafka
